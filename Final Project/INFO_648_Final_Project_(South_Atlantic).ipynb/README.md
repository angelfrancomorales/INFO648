# INFO 648 Final Project - South Atlantic Region 

## Overview
This project predicts neighborhood population growth within the South Atlantic Region (Delaware, Maryland, Florida, District of Columbia, North Carolina, South Carolina, Virginia, West Virginia, and Georgia) using U.S. Census tract data and a CRISP-DM pipeline.

## How to Run
### 1. Clone the repository
git clone: (https://github.com/angelfrancomorales/INFO648/blob/main/Final%20Project/INFO_648_Final_Project_(South_Atlantic).ipynb/INFO_648_Final_Project_(South_Atlantic)%20(4).ipynb)

### 2. Upload data files to Google Colab
Upload these four files:
- students_tracts_raw.csv
- forecast_tracts_2020.csv
- data_dictionary.csv
- region_tract_counts.csv

### 3. Open the notebook
Open INFO_648_Final_Project_(South_Atlantic).ipynb

### 4. Run all cells in order from top to bottom
Runtime>Run all

## Data Source
Data was provided by the professor. Drawn from IPUMS NHGIS time series tables, standardized to 2010 census tract boundaries.

## Dependencies
All libraries are available in Google Colab by default: pandas, scikit-learn, and matplotlib.
